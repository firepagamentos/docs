## Visão Geral

Consulte todas as transações PIX (cobranças e transferências) da sua conta com opções avançadas de filtro, ordenação e paginação.

## Parâmetros de Query

<ParamField query="page" type="integer">
  Número da página (padrão: 1)
</ParamField>

<ParamField query="limit" type="integer">
  Itens por página (padrão: 20, máx: 100)
</ParamField>

<ParamField query="type" type="string">
  Tipo de transação: `cash_in`, `cash_out` ou `all` (padrão)
</ParamField>

<ParamField query="status" type="string">
  Status: `paid`, `waiting_payment`, `expired`, `cancelled`, `refund_in`
</ParamField>

<ParamField query="start_date" type="string">
  Data de início (ISO 8601): `2024-01-01T00:00:00Z`
</ParamField>

<ParamField query="end_date" type="string">
  Data de fim (ISO 8601): `2024-01-31T23:59:59Z`
</ParamField>

<ParamField query="min_value" type="number">
  Valor mínimo em reais
</ParamField>

<ParamField query="max_value" type="number">
  Valor máximo em reais
</ParamField>

<ParamField query="search" type="string">
  Busca por nome, documento ou external_id
</ParamField>

## Exemplo de Requisição

<CodeGroup>
```bash cURL - Todas as Transações
curl -X GET "https://api-gateway.firebanking.com.br/v2/transactions?page=1&limit=50" \
  -H "x-api-key: SUA_CHAVE_DE_API"
```

```bash cURL - Com Filtros
curl -X GET "https://api-gateway.firebanking.com.br/v2/transactions?type=cash_in&status=paid&start_date=2024-01-01T00:00:00Z&end_date=2024-01-31T23:59:59Z" \
  -H "x-api-key: SUA_CHAVE_DE_API"
```

```javascript JavaScript
const transactions = await fetch('https://api-gateway.firebanking.com.br/v2/transactions?page=1&limit=50&type=cash_in', {
  headers: {
    'x-api-key': 'SUA_CHAVE_DE_API'
  }
});

const result = await transactions.json();
console.log(`Total: ${result.total} transações`);
```
</CodeGroup>

## Filtros Avançados

### Por Período
```bash
# Últimos 30 dias
start_date=2024-01-01T00:00:00Z&end_date=2024-01-31T23:59:59Z

# Hoje
start_date=2024-01-15T00:00:00Z&end_date=2024-01-15T23:59:59Z
```

### Por Valor
```bash
# Entre R$ 100 e R$ 1.000
min_value=100&max_value=1000

# Acima de R$ 500
min_value=500
```

### Por Status
```bash
# Apenas pagos
status=paid

# Aguardando pagamento
status=waiting_payment
```

## Ordenação

<ParamField query="sort" type="string">
  Campo para ordenação: `created_at`, `amount`, `status`
</ParamField>

<ParamField query="order" type="string">
  Direção: `asc` ou `desc` (padrão)
</ParamField>

```bash
# Mais recentes primeiro
sort=created_at&order=desc

# Menores valores primeiro
sort=amount&order=asc
```

## Casos de Uso

### Relatório Diário
```javascript
async function relatorioDiario() {
  const hoje = new Date().toISOString().split('T')[0];
  const response = await fetch(
    `https://api-gateway.firebanking.com.br/v2/transactions?start_date=${hoje}T00:00:00Z&end_date=${hoje}T23:59:59Z&status=paid`,
    { headers: { 'x-api-key': 'sua_chave' } }
  );
  
  const data = await response.json();
  console.log(`Hoje: ${data.summary.total_transactions} transações, R$ ${data.summary.total_value / 100}`);
}
```

### Busca por Cliente
```bash
# Buscar por nome ou documento
search=Maria Silva

# Buscar por external_id
search=pedido-12345
```
