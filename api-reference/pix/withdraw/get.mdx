## Visão Geral

Consulte informações detalhadas de uma transferência PIX (cash out) utilizando o ID da transação. Este endpoint retorna dados completos sobre o status, valores, beneficiário e metadados da transferência.

<Info>
  Use o mesmo endpoint para consultar tanto cobranças (cash in) quanto transferências (cash out).
</Info>

## Parâmetros da URL

<ParamField path="id" type="string" required>
  ID da transferência PIX ou `externalId` fornecido na criação
</ParamField>

## Parâmetros de Query

<ParamField query="type" type="string">
  Tipo de busca: `internal` (padrão) ou `external`
</ParamField>

## Status da Transferência

### Estados Possíveis

<AccordionGroup>
  <Accordion title="WAITING_CONFIRMATION">
    - **Descrição**: Transferência criada, aguardando processamento
    - **Duração**: Alguns segundos
    - **Próximo status**: `WITHDRAW_PAID` ou `ERROR`
  </Accordion>

  <Accordion title="WITHDRAW_PAID">
    - **Descrição**: Transferência realizada com sucesso
    - **Valor**: Debitado e creditado no destino
    - **Status final**: Transferência concluída
  </Accordion>

  <Accordion title="ERROR">
    - **Descrição**: Erro no processamento
    - **Valor**: Não debitado ou estornado
    - **Ação**: Verificar erro e tentar novamente
  </Accordion>
</AccordionGroup>
