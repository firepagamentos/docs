---
title: 'Obter Saldo da Conta'
openapi: 'GET /account/balance'
---

Recuperar as informações atuais do saldo da conta, incluindo saldos disponíveis e pendentes. Este endpoint fornece dados de saldo em tempo real para sua conta FireBanking.

## Tipos de Saldo

<CardGroup cols={2}>
  <Card title="Saldo Disponível" icon="money-bill-wave">
    Fundos que podem ser sacados ou usados imediatamente
  </Card>
  <Card title="Saldo Pendente" icon="clock">
    Fundos sendo processados que ficarão disponíveis em breve
  </Card>
</CardGroup>

## Exemplo de Requisição

```bash cURL
curl -X GET "https://api.firebanking.com/v1/account/balance" \
  -H "Authorization: Bearer SEU_TOKEN_DE_NEGOCIO"
```

## Exemplo de Resposta

```json Resposta
{
  "availableBalance": 150000,
  "pendingBalance": 25000,
  "currency": "BRL",
  "lastUpdated": "2024-01-15T16:30:00Z",
  "accountId": "acc_12345",
  "balanceDetails": {
    "settlementBalance": 140000,
    "reserveBalance": 10000,
    "processingFees": 3500
  }
}
```

## Campos da Resposta

<ResponseField name="availableBalance" type="integer">
  Saldo disponível em centavos (pode ser sacado imediatamente)
</ResponseField>

<ResponseField name="pendingBalance" type="integer">
  Saldo pendente em centavos (fundos sendo processados)
</ResponseField>

<ResponseField name="currency" type="string">
  Código da moeda (sempre "BRL" para Real Brasileiro)
</ResponseField>

<ResponseField name="lastUpdated" type="string">
  Timestamp de quando o saldo foi atualizado pela última vez
</ResponseField>

<ResponseField name="settlementBalance" type="integer">
  Fundos de transações liquidadas
</ResponseField>

<ResponseField name="reserveBalance" type="integer">
  Fundos mantidos em reserva para chargebacks/estornos
</ResponseField>

<ResponseField name="processingFees" type="integer">
  Taxas de processamento acumuladas a serem deduzidas
</ResponseField>

## Cálculo do Saldo

### Saldo Disponível
Saldo disponível é calculado como:
```
Disponível = Saldo de Liquidação - Saldo de Reserva - Taxas de Processamento
```

### Saldo Pendente
Inclui:
- **Pagamentos em processamento** que ainda não foram liquidados
- **Autorizações capturadas** aguardando liquidação
- **Estornos** sendo processados de volta aos clientes
- **Transações de ajuste** do processador de pagamentos

## Fundos de Reserva

Uma porção do seu saldo pode ser mantida em reserva para cobrir:
- **Chargebacks potenciais** de transações contestadas
- **Obrigações de estorno** para transações recentes
- **Gerenciamento de risco** baseado na atividade da conta

Percentuais de reserva são determinados por:
- Volume e histórico de transações
- Fatores de risco da indústria
- Taxas de chargeback
- Idade e performance da conta

## Limitação de Taxa

Este endpoint tem os seguintes limites:
- **60 requisições por minuto** por conta
- **1000 requisições por hora** por conta

<Warning>
  Não faça polling excessivo deste endpoint. Para atualizações de saldo em tempo real, considere usar webhooks para eventos de transação.
</Warning>

## Cenários de Erro

- **Conta não encontrada**: Conta inválida ou inativa
- **Permissões insuficientes**: Chave de API não tem acesso ao saldo
- **Conta suspensa**: Conta está temporariamente suspensa
- **Serviço indisponível**: Serviço de saldo está temporariamente indisponível

## Melhores Práticas

1. **Cache dados de saldo** - Evite chamadas desnecessárias à API
2. **Use webhooks** - Seja notificado de eventos que afetam o saldo
3. **Monitore reservas** - Rastreie tendências da taxa de reserva
4. **Planeje saques** - Considere liquidações pendentes
5. **Concilie regularmente** - Compare com contabilidade interna

## Webhooks de Saldo

Inscreva-se nestes webhooks para atualizações de saldo:
- `balance.updated` - Quando o saldo disponível muda
- `settlement.completed` - Quando transações são liquidadas
- `reserve.adjusted` - Quando valores de reserva mudam

## Formato da Moeda

Todos os valores são retornados em **centavos**:
- R$ 1.500,00 = `150000` centavos
- R$ 250,00 = `25000` centavos  
- R$ 35,00 = `3500` centavos

<Tip>
  Para exibir moeda adequadamente, divida por 100 e formate de acordo com as convenções de moeda brasileira.
</Tip>
