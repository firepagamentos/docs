---
title: 'Listar Pagamentos'
openapi: 'GET /credit-card/v1/payment'
---

Recuperar uma lista paginada de pagamentos da sua conta. Este endpoint suporta vários parâmetros de consulta para filtragem e paginação.

## Parâmetros de Consulta

<ParamField query="offset" type="integer" default="0">
  Número de itens a pular para paginação
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Número máximo de itens a retornar (1-100)
</ParamField>

<ParamField query="count" type="boolean" default="false">
  Incluir contagem total na resposta para paginação
</ParamField>

## Estrutura da Resposta

A resposta inclui um array de objetos de pagamento com metadados de paginação quando `count=true`.

## Exemplo de Requisição

```bash cURL
curl -X GET "https://api.firebanking.com/v1/payment?offset=0&limit=10&count=true" \
  -H "x-api-key: SUA_CHAVE_DE_API"
```
