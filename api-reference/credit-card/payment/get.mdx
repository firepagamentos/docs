---
title: 'Obter Pagamento por ID'
openapi: 'GET /payment/{paymentId}'
---

Recuperar informações detalhadas sobre um pagamento específico usando seu identificador único.

## Parâmetros de Caminho

<ParamField path="paymentId" type="string" required>
  O identificador único do pagamento a ser recuperado
</ParamField>

## Exemplo de Requisição

```bash cURL
curl -X GET "https://api.firebanking.com/v1/payment/payment-123" \
  -H "x-api-key: SUA_CHAVE_DE_API"
```

## Valores de Status de Pagamento

- **pending**: Pagamento está sendo processado
- **completed**: Pagamento foi bem-sucedido
- **failed**: Pagamento falhou
- **cancelled**: Pagamento foi cancelado
- **refunded**: Pagamento foi estornado

<Tip>
  Use este endpoint para verificar o status atual de um pagamento e recuperar detalhes completos do pagamento.
</Tip>
