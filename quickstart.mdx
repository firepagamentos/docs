## Configure seu ambiente de desenvolvimento

Aprenda como atualizar sua documentação localmente e implantá-la publicamente.

### Editar e visualizar

<AccordionGroup>
  <Accordion icon="github" title="Clone sua documentação localmente">
    Durante o processo de integração, criamos um repositório no seu Github com
    o conteúdo da sua documentação. Você pode encontrar este repositório no nosso
    [dashboard](https://dashboard.mintlify.com). Para clonar o repositório
    localmente, siga estas
    [instruções](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    no seu terminal.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Visualizar mudanças">
    A visualização ajuda você a garantir que suas mudanças fiquem como pretendido. Construímos uma
    interface de linha de comando para renderizar essas mudanças localmente. 
    1. Instale a
    [Mintlify CLI](https://www.npmjs.com/package/mintlify) para visualizar as
    mudanças da documentação localmente com este comando: ``` npm i -g mintlify ```
    2. Execute o seguinte comando na raiz da sua documentação (onde
    está o `docs.json`): ``` mintlify dev ```
    <Note>
      Se você está atualmente usando o arquivo de configuração legado ```mint.json```, atualize a Mintlify CLI:


      ```npm i -g mintlify@latest```
      E execute o novo comando de upgrade no seu repositório de docs:

      ```mintlify upgrade```
      Você deve estar usando agora o novo arquivo de configuração ```docs.json```. Fique à vontade para deletar o arquivo ```mint.json``` do seu repositório.
    </Note>
  </Accordion>
</AccordionGroup>

### Fazer deploy das suas mudanças

<AccordionGroup>

<Accordion icon="message-bot" title="Instalar nosso app do Github">
  Nosso app do Github automaticamente faz deploy das suas mudanças para o site da documentação, então você
  não precisa gerenciar deployments por conta própria. Você pode encontrar o link para instalar no
  seu [dashboard](https://dashboard.mintlify.com). Uma vez que o bot tenha sido
  instalado com sucesso, deve haver uma marca de verificação próxima ao hash de commit
  do repositório.
</Accordion>
<Accordion icon="rocket" title="Push suas mudanças">
  [Commit e faça push das suas mudanças para o
  Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
  para que suas mudanças sejam atualizadas no site da documentação. Se você fizer push e não ver que
  o app do Github fez o deploy das suas mudanças com sucesso, você também pode atualizar manualmente
  sua documentação através do nosso [dashboard](https://dashboard.mintlify.com).
</Accordion>

</AccordionGroup>
