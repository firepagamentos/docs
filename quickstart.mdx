## Configuração Inicial

### 1. Obtenha suas Credenciais

1. Acesse o [Dashboard FireBanking](https://dashboard.firebanking.com)
2. Vá em **API & Webhooks** → **Chaves de API**
3. Copie sua chave de **Sandbox** (formato: `fbk_test_...`)

```bash Exemplo de Chave
fbk_test_1234567890abcdef1234567890abcdef
```

### 2. Ambientes Disponíveis

<CardGroup cols={2}>
  <Card title="Sandbox (Testes)" icon="flask">
    **Base URL**: `https://api-gateway.firebanking.dev`

    Use para desenvolvimento e testes
  </Card>
  <Card title="Produção" icon="globe">
    **Base URL**: `https://api-gateway.firebanking.com.br`

    Use apenas com dados reais validados
  </Card>
</CardGroup>

### 3. Estrutura da API

Nossa API é **modular** e organizada por método de pagamento:

<AccordionGroup>
  <Accordion title="Endpoints por Método">
    - **PIX**: `{base_url}/pix/v2/charge/qrcode`
    - **Cartão**: `{base_url}/credit-card/v1/payment`
    - **Boleto**: `{base_url}/bank-slip/v1/payment`
  </Accordion>

  <Accordion title="Características Técnicas">
    - **Autenticação**: Header `x-api-key` obrigatório
    - **Formato JSON**: Todas as requisições e respostas
    - **Códigos HTTP**: Semânticos (200, 400, 403, 500, etc.)
    - **IDs únicos**: Identificadores consistentes
    - **Webhooks**: Notificações em tempo real
  </Accordion>
</AccordionGroup>

## Teste sua Conexão

Verifique se sua autenticação está funcionando:

<CodeGroup>
```bash cURL
curl -X GET "https://api-gateway.firebanking.dev/v1/account/balance" \
  -H "x-api-key: fbk_test_SUA_CHAVE_AQUI" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api-gateway.firebanking.dev/v1/account/balance', {
  headers: {
    'x-api-key': 'fbk_test_SUA_CHAVE_AQUI',
    'Content-Type': 'application/json'
  }
});

const balance = await response.json();
console.log('Conta ativa:', balance.account_status === 'active');
```

```python Python
import requests

response = requests.get(
    'https://api-gateway.firebanking.dev/v1/account/balance',
    headers={
        'x-api-key': 'fbk_test_SUA_CHAVE_AQUI',
        'Content-Type': 'application/json'
    }
)

balance = response.json()
print('Conta ativa:', balance['account_status'] == 'active')
```
</CodeGroup>

**Resposta esperada:**
```json
{
  "balance": 0,
  "currency": "BRL",
  "account_status": "active"
}
```

## Guias de Integração por Método

Agora que você tem suas credenciais configuradas, escolha seu método preferido e siga o guia específico:

### 🚀 Recomendado para Iniciantes: PIX

<Card title="Guia Completo PIX" icon="qrcode" href="/guides/integration/pix">
  **Por que começar com PIX?**
  - ✅ Integração mais simples
  - ✅ Menor custo (taxa fixa)
  - ✅ Confirmação instantânea
  - ✅ Funciona 24h/7 dias

  **Tempo estimado**: 15-20 minutos
</Card>

### 💳 Para E-commerce: Cartão de Crédito

<Card title="Guia Completo Cartão" icon="credit-card" href="/guides/integration/credit-card">
  **Ideal se você precisa de:**
  - ✅ Parcelamento em até 12x
  - ✅ Assinaturas recorrentes
  - ✅ Tokenização de cartões
  - ✅ Pré-autorização

  **Tempo estimado**: 25-30 minutos
</Card>

### 🏦 Para Valores Altos: Boleto Bancário

<Card title="Guia Completo Boleto" icon="file-invoice" href="/guides/integration/bank-slip">
  **Ideal se você tem:**
  - ✅ Clientes sem cartão
  - ✅ Valores acima de R$ 200
  - ✅ Necessidade de comprovante físico
  - ✅ Flexibilidade de prazo

  **Tempo estimado**: 20-25 minutos
</Card>
