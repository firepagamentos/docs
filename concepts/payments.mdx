## Visão Geral

O módulo de cartão de crédito do FireBanking foi projetado para oferecer uma integração **simples, direta e previsível** para processamento de transações de cartão de crédito. O sistema permite transações pontuais ou parceladas, com políticas de taxas configuráveis.

## Características Principais

### Módulo Dedicado
- Foco exclusivo em transações de cartão de crédito
- Separação clara das funcionalidades de outros módulos de pagamento
- Controle de saldo independente de outras modalidades

### Flexibilidade de Pagamento
- **Pagamento à vista** ou **parcelamento** até 24 vezes
- Políticas de taxa configuráveis:
  - Absorvidas pelo estabelecimento comercial
  - Repassadas para o cliente final

### Gestão Simplificada
- Não é necessário pré-criar perfis de compradores
- Contrato JSON único para detalhes da transação
- Identificadores internos e externos para cada transação

## Fluxo de Liquidez

O FireBanking garante **liquidez previsível** com:
- Transferência de fundos para conta Pix em **D+2**
- Transações elegíveis para antecipação de recebíveis
- Transparência total no cronograma de repasses

## Segurança e Integridade

### Autenticação
- Utilização obrigatória do header `x-api-key`
- Recomendação de whitelist de IPs
- Gestão cuidadosa do ciclo de vida das chaves de API

### Rastreabilidade
- Sistema de identificadores duplos (interno/externo)
- Logs completos de transações
- Auditoria detalhada de operações
